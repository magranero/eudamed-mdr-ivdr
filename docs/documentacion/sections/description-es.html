EUDAMED MDR/IVDR Device Catalogue Ontology: Descripción volver a índice

Ontología semántica para el catálogo europeo de dispositivos médicos de alto riesgo (MDR, Regulation EU 2017/745) y de dispositivos de diagnóstico in vitro (IVDR, Regulation EU 2017/746). Modela dispositivos, fabricantes, clases de riesgo, riesgos identificados según IMDRF, incidentes notificados y acciones correctivas de seguridad en campo (FSCA) tal y como se registran en la base de datos EUDAMED.
